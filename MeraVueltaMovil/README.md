# MeraVuelta Movil

Aplicacion movil (React Native + Expo) para operacion de domiciliarios y seguimiento de entregas.

## Stack

- React Native 0.70 + Expo 47
- Firebase
- Geolocalizacion (`expo-location`, `expo-task-manager`)

## Comandos

```bash
npm install
npm run start
npm run android
npm run ios
```

## Configuracion

- Variables de entorno en `.env`.
- Requiere servicios backend activos (`MeraVueltaApi`) para operacion completa.
